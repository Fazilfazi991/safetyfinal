# Safety World – Website

Vite + React plain JavaScript. Zero TypeScript, zero Manus deps. Deploys to Vercel instantly.

## Local dev
```bash
npm install
npm run dev
```

## Deploy to Vercel via GitHub
```bash
git init
git add .
git commit -m "initial commit"
git remote add origin https://github.com/YOUR_USER/safetyworld.git
git push -u origin main
```
Then vercel.com/new → Import → Deploy. Done.
